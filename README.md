### brick架构技术栈

* 目前基于JDK1.8 上搭建
* 基础架构
  * Redisson
  * FastJson
  * Beetl
  * Smart-doc
  * Logback
  * SpringBoot
  * SpringCloud
    * Gateway
    * OpenFeign
  * MybatisPlus
  * Duid
  * Sharding-JDBC
  * Commons 组件
  * Sa-Token
* 中间件
  * Redis
  * Nacos
  * Sentinel
  * Seata
  * Nginx
  * Minio
  * RocketMQ
  * Kafka
  * Flume
  * Elestic-Job

### brick 方向

> 为真**零**代码开发平台而努力, 基于模块化可插拔的方式管理系统架构.。。。
